왜 gnFortress인가

단순 스캐너가 아닙니다. 인증을 아는 보안 파트너입니다.

ISMS-P 최신

인증 기준 전문 매핑

KISA ISMS-P 인증기준을 비롯한 다수의 공인 보안 가이드를 정밀 매핑한 엔진. 글로벌 툴이 모르는 한국 인증 기준에 특화돼 있습니다.

생명주기 추적

재발까지 추적해 감사 증빙

미조치 → 조치완료 → 재발을 한 줄로 추적. "한 번 고쳤다 또 터진 항목"까지 잡아내 심사 증빙이 완벽합니다.

뮤트 규칙

오탐은 음소거, 진짜 위험에 집중

의도된 예외·오탐은 뮤트해 노이즈를 줄이고, 실제 조치가 필요한 항목에만 집중하게 합니다.

금융권 검증

국내 금융권 1위의 실전 엔진

4,000개 이상의 클라우드 워크로드를 지켜온 가디언넷의 검증된 보안 역량이 그대로 담겼습니다.

스케줄 점검

벼락치기 끝, 상시 점검

원하는 주기로 자동 점검해 상시 인증 대비 태세를 유지합니다. 심사 전날 야근이 사라집니다.

멀티 계정

50~500 계정 엔터프라이즈 설계

대규모 멀티 계정 환경을 한 화면에서. 계정·서비스별 컴플라이언스 현황을 통합 관리합니다.

전문가 진단 · 대규모 도입

직접 연결이 부담되시거나, 대규모(금융·공공)이신가요?

전문가가 무료로 진단하고 데모를 드립니다. 보안성 검토·계약·온프렘·SLA·관제(XDR)까지 엔터프라이즈 요건을 함께 설계합니다. 셀프서비스가 부담되는 첫 도입에도 적합합니다.

전문가 진단·데모 신청

영업 부담 없이 리포트만 받아보셔도 됩니다.

자주 묻는 질문

시작 전 궁금하신 점

정말 무료로 시작할 수 있나요?

네. Starter는 AWS 1계정을 무료로 직접 점검하고 리포트까지 생성할 수 있습니다. 신용카드도 필요 없습니다.

우리 클라우드에 안전한가요?

점검은 고객님이 직접 본인 AWS 계정에 배포한 읽기 전용(read-only) IAM 역할로만 동작합니다. 설정 변경이 불가능하고, 본인 소유 계정만 연결할 수 있어 안전합니다.

AWS 연결이 어렵지 않나요?

제공되는 CloudFormation 템플릿으로 버튼 한 번에 읽기전용 역할이 생성됩니다. 가이드를 따라 2~3분이면 됩니다.

티어별로 뭐가 다른가요?

실행 범위가 다릅니다. Starter는 1계정, Professional은 최대 10계정 + HostIPS 5워크로드, Enterprise는 무제한입니다. 자세한 내용은 요금제를 참고하세요.

결제는 어떻게 하나요?

Professional은 카드와 세금계산서·계좌이체를 지원합니다. Enterprise는 계약·견적으로 진행합니다.

직접 하기 부담스러우면요?

"전문가 진단"을 신청하시면 가디언넷 전문가가 진단·데모를 도와드립니다. 금융·공공 등 대규모 도입에 적합합니다.

지금 우리 클라우드, 인증 통과할 수 있을까?

추측하지 마세요. 무료로 가입해 직접 점수부터 확인하세요.
