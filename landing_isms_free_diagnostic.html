자주 묻는 질문

진단 전 궁금하신 점

정말 무료인가요?

네. AWS 계정 1개에 대한 ISMS-P 진단과 점수 리포트를 무료로 제공합니다. 별도 비용이나 결제 정보는 필요하지 않습니다.

우리 클라우드에 안전한가요?

안전합니다. 읽기 전용(read-only) IAM 역할만 사용하므로 설정 변경이 불가능하며, 진단 외 목적의 데이터 반출도 일절 없습니다. 점검 항목과 권한 범위는 사전에 모두 공유드립니다.

결과는 얼마나 걸리나요?

신청 후 연결이 완료되면 통상 1영업일 이내에 점수 리포트를 보내드립니다.

꼭 도입해야 하나요? 영업 전화가 부담스럽습니다.

전혀 부담 갖지 않으셔도 됩니다. 리포트만 받아보고 내부 검토용으로 활용하셔도 좋습니다. 도입은 전적으로 고객님의 선택입니다.

AWS 외 다른 클라우드도 되나요?

현재 무료 진단은 AWS를 지원합니다. GCP·Azure·NHN Cloud는 순차 확장 예정이며, 도입 상담 시 로드맵을 안내드립니다.
